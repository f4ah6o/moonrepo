## 作業メモ

- root `justfile` に `cloudflare-op-create`、`cloudflare-op-env`、`cloudflare-build`、`cloudflare-dev`、`cloudflare-deploy-preview`、`cloudflare-deploy` を追加した。
- 対応 repo は `domainprocessschema.mbt` と `vizprocess.mbt` のみに固定し、それ以外は即時エラーにした。
- `domainprocessschema.mbt` では既存 Worker 構成を再利用し、production custom domain として `domainprocessschema.f12o.com` を Wrangler route に追加した。
- `vizprocess.mbt` では browser demo をそのまま公開 asset として配る方針にし、新しい API Worker は作らず `env.ASSETS.fetch(request)` だけを返す最小 Worker にした。
- `vizprocess.mbt` に `package.json`、`wrangler.jsonc`、`examples/browser-demo/worker.mjs`、`examples/browser-demo/.env.opz.example`、`scripts/build-cloudflare-demo.mjs` を追加した。
- asset bundle は `_build/cloudflare/browser-demo/` に `index.html`、`prefetch.mjs`、`duckdb-loader.mjs`、`opfs.mjs`、`wasm.js`、`fixtures/datasets/sales.csv` を出す。
- `index.html` 内の `../../_build/js/release/build/wasm/wasm.js` と `../../fixtures/datasets/sales.csv` は deploy 用 bundle で `./wasm.js` と `./fixtures/datasets/sales.csv` に書き換えるようにした。
- 検証では `moon -C repos/vizprocess.mbt build --target js --release` と `node repos/vizprocess.mbt/examples/browser-demo/node-smoke.mjs` を通す。
- `domainprocessschema.mbt` / `vizprocess.mbt` とも production route は top-level ではなく `env.production.routes` に置いた。こうしておくと `preview` deploy が custom domain を奪わず、`workers.dev` だけで確認できる。
- `vizprocess.mbt` の `compatibility_date` は Cloudflare API が future date と見なさないよう `2026-05-07` に合わせた。
- `just cloudflare-build domainprocessschema.mbt`、`just cloudflare-build vizprocess.mbt`、`just cloudflare-deploy-preview <repo>`、`just cloudflare-deploy <repo>` は通った。
- preview URL:
  - `https://domainprocessschema-wasm-demo-preview.f12o.workers.dev`
  - `https://vizprocess-browser-demo-preview.f12o.workers.dev`
- production URL:
  - `https://domainprocessschema.f12o.com`
  - `https://vizprocess.f12o.com`
- headless browser では `vizprocess.f12o.com` の初回 render、DuckDB example、OPFS seed、OPFS example render を確認した。
- `domainprocessschema` は preview / production とも公開ページが開き、`Start runtime session` と `Current view` が見えるところまで確認した。
