## 学び

`dwiki` は repo の全体像を先に掴むには便利だが、生成された概要や索引済み情報は実際の作業時点の local clone とずれる可能性がある。  
そのため、skill では「低トークンな入口」として使い、編集や結論の根拠は必ず local file で確認する流れにした。
