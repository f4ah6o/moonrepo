## 学び・メモ
- MoonBit の PBT は `@qc.quick_check(@qc.forall(...))` の形で小さく導入できる。
- 最初の導入では generator の文字種を制限すると flaky を抑えやすい。
- 重複 key のような仕様は、example test より property で網羅した方が回帰を防ぎやすい。
